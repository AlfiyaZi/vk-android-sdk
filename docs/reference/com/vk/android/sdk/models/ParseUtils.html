public static T parseViaReflection (T object, JSONObject source)

Parses object with follow rules: 1. All fields should had a public access. 2. The name of the filed should be fully equal to name of JSONObject key. 3. Supports parse of all Java primitives, all java.lang.String, arrays of primitive types, java.lang.Strings and VKModels, list implementation line VKList, VKAttachments.VKAttachment or VKPhotoSizes, VKModels. 4. Boolean fields defines by vk_int == 1 expression.
