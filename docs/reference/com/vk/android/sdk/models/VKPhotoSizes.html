Class Overview

Model to parse a list of photo with photo sizes format. When photo_sizes=1 parameter is used in methods, response contains an info about original photo copies with different sizes in sizes array with objects, each of them contains following fields:

  • src — url of image copy;
  • width — copy width, px;
  • height — copy height, px;
  • type — notation for copy size and ratio.
